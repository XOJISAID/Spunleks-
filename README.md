# SpunLeks Homepage

This is the official landing page for **SpunLeks**, a cotton manufacturing factory.

## Stack
- **Next.js 14+** (App Router)
- **Tailwind CSS** (Styling)
- **Framer Motion** (Animations)
- **Lucide React** (Icons)

## Features
- **Animated Loader**: Shows the SpunLeks logo animation on initial visit.
- **Hero Section**: Responsive banner.
- **About & Features**: Information about production.
- **Contact**: Footer with contact details.

## Getting Started

1. Install dependencies:
   `ash
   npm install
   `

2. Run the development server:
   `ash
   npm run dev
   `

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Customization
- Place your actual logo in \src/components/SpunLeksLogo.tsx\ or replace the SVG.
- Update content texts in the respective components in \src/components/\.
